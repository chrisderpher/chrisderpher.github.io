# ouroboros

Multiplication speed drill (0-10 x 0-10) on a fixed circular track of 12 problems. Problems expire faster as level increases. Catching the oldest unanswered tail before it expires awards a bonus.

## Controls

- 0-9 - enter digits
- Enter - submit
- Backspace - delete digit
- ESC - pause
- SPACE - start / return to menu
- ? - help

High scores persist in LocalStorage.
