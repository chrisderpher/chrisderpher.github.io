Automating Humble Choice Steam Redemptions With Playwright

Download: Download the public script package

This page hosts a supervised Playwright automation script for users who want help redeeming Steam keys they already received from their own Humble Choice account.

Safety and account-use boundaries

This project is only for users redeeming games from their own Humble and Steam accounts. It does not bypass payment, logins, Steam Guard, captchas, ownership checks, regional limits, rate limits, or account restrictions. Run it only while you are present and supervising the browser session.

  • You sign in to Humble and Steam yourself.
  • You approve any Steam Guard, captcha, or account prompt yourself.
  • The script should stop whenever a prompt needs human judgment.
  • You remain responsible for following Humble Bundle, Steam, and publisher terms.

What the automation is meant to do

The script is designed to reduce repetitive clicking after you have already purchased Humble Choice and are redeeming keys into your own Steam library. Playwright opens a normal browser session, lets you authenticate normally, and can help move through owned-key redemption pages while leaving account-sensitive actions visible and supervised.

Suggested workflow

  1. Download and extract the package.
  2. Read the included README and review the script before running it.
  3. Install dependencies in the extracted folder.
  4. Start the automation and keep the browser visible.
  5. Complete logins, Steam Guard, captchas, confirmations, or errors manually.

Example commands

After extracting the zip, the typical setup flow for a Playwright project looks like this:

npm install
npx playwright install

Then run the script using the command documented in the package README. For example, if the package exposes a start script:

npm start

Supervision checklist

  • Keep the browser window visible.
  • Do not leave the automation running unattended.
  • Pause or stop the run if Steam or Humble shows an unexpected warning.
  • Verify each redemption result in your Steam library or account history.
  • Do not share credentials, cookies, session files, or redeemed keys.
