# fractions

Imperial tape-measure fraction practice. Eight drill modes: ordering, bigger/smaller, to decimal, between marks, add/subtract, mixed to improper, difference, inches to feet.

## Controls

- SPACE - start / return to menu
- 1-8 - select mode
- 1-5 / arrows / digits - mode-specific answer input
- Enter - submit
- Backspace - clear
- Tab - switch field (inches to feet)
- ESC - pause
- ? - help

High scores persist in LocalStorage.
